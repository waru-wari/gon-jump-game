📋 ภาพรวม

empeo Land Adventure เป็นเกมออนไลน์ฟรีที่พัฒนาโดยทีม empeo เราเก็บข้อมูลน้อยที่สุดเท่าที่จำเป็นสำหรับการแสดง ลีดเดอร์บอร์ดออนไลน์ เท่านั้น ไม่มีบัญชีผู้ใช้ ไม่มีการเข้าสู่ระบบ และไม่มีการขายข้อมูล

📦 ข้อมูลที่เราเก็บ

เฉพาะเมื่อคุณกด "ส่งคะแนน"
ข้อมูล รูปแบบที่เก็บ วัตถุประสงค์
อีเมล ค่าแฮช SHA-256 (เข้ารหัส ไม่สามารถย้อนกลับได้) · แสดงในรูปบลอค เช่น j**n@gofive.co.th ระบุตัวตนในลีดเดอร์บอร์ด / อัปเดตคะแนนสูงสุด
ชื่อ (ชื่อเล่น) ข้อความที่คุณพิมพ์ (ไม่เกิน 20 ตัวอักษร) แสดงในลีดเดอร์บอร์ด
คะแนนสูงสุด ตัวเลขจำนวนเต็ม จัดอันดับ
เลเวลสูงสุด ตัวเลข 1–6 แสดงในโปรไฟล์
เวลาที่ส่งคะแนน Unix timestamp (UTC) ลีดเดอร์บอร์ดรายวัน
การยินยอม (Consent) Timestamp ที่กดยินยอม หลักฐาน PDPA
💡 อีเมลของคุณถูกเข้ารหัสทันที ก่อนจะบันทึกลงเซิร์ฟเวอร์ เราไม่สามารถดู แชร์ หรือกู้คืนอีเมลต้นฉบับได้เลย

🚫 ข้อมูลที่เราไม่เก็บ

  • อีเมลต้นฉบับ (Plaintext email)
  • ที่อยู่ IP อย่างถาวร (ใช้เฉพาะชั่วคราวเพื่อป้องกันการโกง — หมดอายุใน 60 วินาที)
  • User Agent / ข้อมูลเบราว์เซอร์
  • ตำแหน่งที่ตั้ง
  • Cookie / Tracking pixel
  • ข้อมูลส่วนบุคคลอื่นๆ

ข้อมูลระยะเวลา
คะแนนในลีดเดอร์บอร์ดโกลบอลไม่มีกำหนด (จนกว่าจะขอลบ)

✅ สิทธิ์ของคุณ (PDPA)

ภายใต้ พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 คุณมีสิทธิ์:

  • ขอดูข้อมูล — รู้ว่าเราเก็บอะไรเกี่ยวกับคุณ
  • ขอลบข้อมูล — ลบคะแนนและโปรไฟล์ทั้งหมดออกจากระบบ
  • แก้ไขชื่อ — เปลี่ยนชื่อที่แสดงในลีดเดอร์บอร์ด
  • ถอนความยินยอม — หากถอนยินยอม ข้อมูลของคุณจะถูกลบทันที

หัวข้ออีเมล: "PDPA Request — empeo Land"
เราจะดำเนินการภายใน 30 วัน

🌏 เซิร์ฟเวอร์และการโอนข้อมูลข้ามพรมแดน

ข้อมูลถูกประมวลผลและเก็บบน Upstash Redis (Singapore, sin1) ผ่านโครงสร้างพื้นฐานของ Vercel ซึ่งตั้งอยู่ใน Singapore AWS Region (ap-southeast-1)

การโอนข้อมูลไปต่างประเทศนี้เป็นไปตาม PDPA มาตรา 29 เนื่องจากประเทศสิงคโปร์มี มาตรฐานคุ้มครองข้อมูลส่วนบุคคลที่เพียงพอภายใต้ PDPA Act 2012

  • ไม่มีการส่งข้อมูลไปยังประเทศอื่นนอกจากสิงคโปร์
  • ไม่มีการแชร์ข้อมูลกับบุคคลที่สาม
  • ไม่มีการขายข้อมูล
